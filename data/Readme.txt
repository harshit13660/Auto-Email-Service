#-----------------------------------------Auto Email Certifiacate Distribution Service------------------------------------#


Auto Email Certifiacate Distribution Service is a multipurpose Email sending service that can send mails in bulk quantity without any fault.
You can select your own desired certificate which you want to send in bulk members, with your desired details in it, like Name course details etc,
This Application just need an .xlxs file with [ Name, Course, Email ] in it. 
--------------------------------------------------Simple and Effective-----------------------------------------------------

Requirements:

* One Excel file with complete data.
* Desired certificate that you want to send.
* Email and Password with Allow Access to less secure apps ----Most Important-------

------------------------------Guide For allowing access to Less secure apps-----------------------------------------------

HOW TO DO IT?.

1.Login to Gmail through Windows, using Internet Explorer.
2.Once signed in, navigate to: https://www.google.com/settings/security/lesssecureapps
3.Select the radio button to allow "Less Secure Apps" on.
4.Sign out of Gmail, restart Guide and select 1 from the Main Menu for Emails. Emails should now be working.


-----------------------------------------------------------------------------------------------------------------------------
						How to operate Software?

1. Upload certificate from Top Left Corner.
2. Adjust Text where you want to display it on certificate.
3. Preview Certificate by clicking on button.
4. Upload Excel file.[Keep The format as -----( Name, Course, Email ) Respectively-----------.
5. Preview file from Bottom Right corner.
6. Finally Click on Send button.
7. In new window, Enter your Email and password with which you want to send email to People [ Make sure to Allow access to
   Less Secure Apps first ], click Login.
8. Enter Subject and Content Body, click Update.
9. After Checking All details, finally click ok on prompt.
10. Wait for process to complete.
